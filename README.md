# Delta Force Collectibles Gunpack

A gunpack for **Timeless and Classics Zero**, adding 3D versions of *Delta Force* collectibles to Minecraft.
**一个为【永恒枪械工坊零】制作的枪包，在 Minecraft 中添加《三角洲行动》的 3D 收集品。**

---

## Installation / 安装

**English:** Download [Timeless and Classics Zero](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero) (or [GitHub](https://github.com/MCModderAnchor/TACZ)), then put this gunpack into `.minecraft/tacz/`

**中文:** 下载模组本体 [永恒枪械工坊零](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero)（或 [GitHub](https://github.com/MCModderAnchor/TACZ)），然后将本枪包放入 `.minecraft/tacz/`
