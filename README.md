# Delta Force Collectibles Gunpack

A gunpack for **Timeless and Classics Zero**, adding 3D versions of *Delta Force* collectibles to Minecraft.

**一个为【永恒枪械工坊零】制作的枪包，在 Minecraft 中添加《三角洲行动》的 3D 收集品。**

---

## Features / 特性

- 3D models of Delta Force collectibles
- Compatible with Timeless and Classics Zero

- 三角洲行动收集品的 3D 模型
- 兼容永恒枪械工坊零

---

## Installation / 安装

1. Install [Timeless and Classics Zero](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero) (or [GitHub](https://github.com/MCModderAnchor/TACZ))
2. Drop this gunpack into `.minecraft/config/tacz/custom_gunpack/`
3. Enjoy!

1. 安装【永恒枪械工坊零】（[CurseForge](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero) | [GitHub](https://github.com/MCModderAnchor/TACZ)）
2. 将本枪包放入 `.minecraft/config/tacz/custom_gunpack/` 文件夹
3. 开始游戏！
